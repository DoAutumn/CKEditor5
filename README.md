## CKEditor5

CKEditor5与Angular集成，并实现插入ECharts图表功能。

### 开发

- npm i
- 下载 [DecoupledEditor.zip](DecoupledEditor.zip) 并解压替换至 node_modules/@ckeditor/ckeditor5-build-decoupled-document/ 目录下
- npm run start

- cd test-server
- node app.js
